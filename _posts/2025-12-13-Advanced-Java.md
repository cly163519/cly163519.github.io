## GET 和 POST 的区别

GET 请求把数据放在 URL 里，POST 请求把数据放在请求体里。
```java
// 这是 Java 代码块
URL url = new URI("http://localhost/test.php").toURL();
HttpURLConnection conn = (HttpURLConnection) url.openConnection();
```
